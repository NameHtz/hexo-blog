Linux 命令

1. 文件和目录

1.1 cd 命令，用于切换当前目录，它的参数是要切换到的目录的路径，可以是绝对路径，也可以是相对路径。

1.2 pwd 命令，显示工作路径

1.3 ls 命令，查看文件与目录的命令，list之意

1.4 cp 命令，用于复制文件，copy之意，它还可以把多个文件一次性地复制到一个目录下

1.5 mv 命令，用于移动文件、目录或更名，move之意

1.6 rm 命令，用于删除文件或目录，remove 之意

2. 查看文件内容

2.1 cat 命令，用于查看文本文件的内容，后接要查看的文件名，通常可用管道与more和less一起使用

3. 文件搜索

3.1 find 命令，用来查找系统的

4. 文件的权限 - 使用 “+” 设置权限，使用 “-“ 用于取消

4.1 chmod 命令，改变文件/文件夹权限

4.2 chown 命令，改变文件的所有者

4.3 chgrp 命令，改变文件所属用户组

5. 文本处理

5.1 grep 命令，分析一行的信息，若当中有我们所需要的信息，就将该行显示出来，该命令通常与管道命令一起使用，用于对一些命令的输出进行筛选加工等等

5.2 paste 命令

5.3 sort 命令

5.4 comm 命令

6. 打包和压缩文件

6.1 tar 命令，对文件进行打包，默认情况并不会压缩，如果指定了相应的参数，它还会调用相应的压缩程序（如gzip和bzip等）进行压缩和解压

6.2 压缩：tar -jcv -f filename.tar.bz2 (要被处理的文件或目录名称) 查询：tar -jtv -f filename.tar.bz2 解压：tar -jxv -f filename.tar.bz2 -C (欲解压缩的目录)

7. 系统和关机（关机、重启和登出）

8. 进程相关的命令

8.1 jps 命令，显示当前系统的java进程情况，及其id号 jps (Java Virtual Machine Process Status Tool) 是JDK 1.5提供的一个显示当前所有java进程pid的命令，简单实用，非常适合在linux/unix平台上简单察看当前java进程的一些简单情况。

8.2 ps 命令，用于将某个时间点的进程运行情况选取下来并输出，process 之意

8.3 kill 命令,用于向某个工作（jobnumber）或者是某个PID（数字）传送一个信号，它通常与ps和jobs命令一起使用

命令格式 : kill[命令参数][进程id]
