那些组件库

React：

  1. antd 是基于 Ant Design 设计规范实现的 高质量 React 组件库

  2. MATERIAL-UI React 组件用于更快速、更简便的 web 开发。

Vue

  1. Vuetify 是 Vue.js 的头号组件库，自 2016 年以来一直在积极开发。

  2. Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

  3. Mint UI 基于 Vue.js 的移动端组件库

  4. PrimeVue

  5. BootstrapVue

简单列举几个；在 Google 上搜索关键字 React｜Vue component library 会获取到更多有用的数据;
